About dotLastFm

dotLastFm is simple .NET wrapper for last.fm API.

NuGet package:

http://nuget.org/packages/dotLastFm

Implemented Last.fm methods:

  • Album.getInfo
  • Album.getTopTags
  • Artist.getInfo
  • Artist.getTopTags
  • Tag.getInfo
  • Tag.getSimilar
  • Tag.getTopTags
  • Tag.getTopAlbums
  • Tag.getTopArtists
  • Tag.getTopTracks
  • Track.getInfo
  • Track.getTopTags
  • Track.getSimilar

Libraries & Tools

This project is using:

  • RestSharp library: https://github.com/restsharp/RestSharp
  • xUnit testing library: http://xunit.codeplex.com/
  • NSubstitute mocking framework: http://nsubstitute.github.com/
  • Psake build automation tool: https://github.com/JamesKovacs/psake
  • StyleCop source analyzer: http://stylecop.codeplex.com/

Requirements

  • .NET Framework 4.0
  • PowerShell for running Psake
